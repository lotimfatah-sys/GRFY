Tutorial

HOW TO CONNECT

Choose your device — setup takes only a few steps, no root or jailbreak required.

PowerTunnel (Android)
Recommended for Android users
1
Install PowerTunnel APK from the official source
2
Open app → tap menu ≡ → tap the host settings icon
3
Paste the URL below, set update period to "On Start"
https://gtpshost.com/raw/GTRY-GH
4
Tap Connect then open Growtopia — done!
Virtual Host (Android)
Alternative method for Android
1
Install Virtual Host APK
2
Open app → tap SELECT HOSTS FILE
3
Choose "Download hosts from URL" then paste:
https://gtpshost.com/d/GTRY-GH
4
Tap Start → open Growtopia — done!
Surge 5 (iOS)
For iPhone and iPad users
1
Download Surge 5 from the App Store
2
Tap OK → Default.conf → IMPORT → Download Profile from URL
3
Paste the config URL:
https://gtpshost.com/ios/GTRY-GH
4
Tap SETUP → Allow VPN → open Growtopia — done!
Windows PC
Desktop setup via hosts file
1
Press Win+R and paste this path:
C:\Windows\System32\drivers\etc
2
Right-click "hosts" → Open with Notepad (Run as Admin)
3
Add this line at the bottom:
15.235.166.218 www.growtopia1.com
4
Save the file then open Growtopia — done!
Linux / Mac
Terminal-based setup
1
Open Terminal and run:
C:\Windows\System32\drivers\etc
2
Add this entry at the bottom:
15.235.166.218 www.growtopia1.com
3
Press Ctrl+O to save, Ctrl+X to exit
4
Flush DNS then launch Growtopia

FAQ

PERTANYAAN UMUM

Quick answers to frequently asked questions about GTRY.

Gimana cara connect ke GTRY?
Pilih device kamu di bagian Tutorial — ada panduan untuk Android (PowerTunnel / Virtual Host), iOS (Surge 5), Windows, dan Linux/Mac. Setup cuma butuh beberapa langkah, tanpa root atau jailbreak.
GTRY gratis atau bayar?
Sepenuhnya gratis untuk dimainkan! Shop tersedia untuk item premium opsional yang mendukung biaya server.
Akun saya hilang, bisa di-recover?
Bisa! Buka ticket di Discord via channel #support. Sertakan bukti kepemilikan akun untuk proses yang lebih cepat.
Apakah trading di sini aman?
Ya! Sistem trading dilindungi anti-scam protection. Gunakan trade in-game dan report scammer ke tim support.
Apa yang bikin GTRY beda dari GTPS lain?
GTRY punya custom items eksklusif, sistem Daily Rewards, komunitas aktif, server 100% uptime, dan tim support responsif. Kami rutin update konten dan event baru setiap bulan.
Berapa lama respon dari tim support?
Tim kami biasanya merespon dalam 1-6 jam. Untuk masalah urgent, gunakan channel #emergency di Discord.
Ada rules yang harus diketahui?
Ya! Dilarang: cheating, hacking, scamming, spam, dan konten tidak pantas. Pelanggaran bisa mengakibatkan ban permanen. Cek halaman Rules untuk daftar lengkap.

BUTUH BANTUAN?

Tim support kami siap membantu lewat Discord, WhatsApp, atau langsung di server.

Ada masalah atau perlu recovery akun?

Cara tercepat untuk mendapat bantuan adalah lewat Discord kami. Buka ticket di channel #support dan tim kami akan segera merespons.
